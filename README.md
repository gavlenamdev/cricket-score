# cricket score
shows live cricket score as notificaion

### Requirement
1. python3
2. python3-notify2
3. python3-bs4
4. libdbus-glib-1-dev libdbus-1-dev (python3-dbus dependency)
5. python3-dbus

### changes to main.py file
in `main.py`, modify the team name in line 7, 8.

### How to run
python main.py
